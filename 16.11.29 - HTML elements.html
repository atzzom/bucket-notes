List of HTML elements

The following is the list of the HTML elements according to the World Wide Web Consortium (W3C).

HTML elements by function

1. Root element

html: root element. Represents the root of the document.

2. Document metadata

Metadata provides information about the page like styles, scripts and data that help software as browsers, search engines, etc. to use and render the page.

<base>: this will serve as the base for all relative URL contained within the document. There only can be one <base> element in the document.

<head>: document metadata container. Collects the document's metadata.

HTML elements by alphabetical order
