This is a snipet of the CSS code used:

code {
background: #2db34a;
border-radius: 6px;
color: #fff;
display: block;
font: 14px/24px "Source Code Pro", Inconsolata, "Lucida Console", Terminal, "Courier New", Courier;

}
header, section, article, aside, footer {margin: 0 1.5% 24px 1.5%;}
footer {margin-bottom: 0;}

A few comments on this CSS code:

- The background color (#2db34a) is green.
- The display: block; property makes the <code> elements be displayed from side to side of the screen.
- The article element is nested into the <section> element.
- Although the paragraphs elements are inside the <article> element the green box render is only assigned to the <code> elements. This means that the <code> boxes do not represent the actual size of its parents elements (header, section, article, aside, footer).
