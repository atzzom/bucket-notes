This script illustrate the declaration of an array element and the initialization of values on it using indexes:

  1. var myArray = new Array();
  2. myArray[0] = "Bob";
  3. myArray[0] = "Pete";
  4. myArray[0] = "Paul";
  6. document.write("myArray[0] = " + myArray[0] + "<br>");
  7. document.write("myArray[2] = " + myArray[2] + "<br>");
  8. document.write("myArray[1] = " + myArray[1] + "<br>");
  10. myArray[1] = "Mike";
  11. document.write("myArray[1] changed to " + myArray[1]);
